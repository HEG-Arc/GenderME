Madame? Monsieur? Madame, Monsieur? Ne vous posez plus la question! Avec GenderME, vous pouvez adresser vos clients avec la bonne formule de politesse.

GenderME c'est plus de 48'000 prénoms, en trois langues, utilisés partout en Europe.

Comment utiliser GenderME?

Nous mettons à votre disposition notre API REST pour vous permettre de mettre vos fichiers clients à jour en fonction de leur prénom. Pour chaque prénom dont vous ne connaissez pas le genre, envoyez une requête à notre service et obtenez une réponse en une fraction de seconde.

API

Notre API est très simple à utiliser. Vous faites une requête GET avec le prénom à chercher et nous vous répondons par un objet JSON comprenant le prénom demandé, son genre ainsi que la langue la plus utilisée par les personnes portant ce prénom. Si le prénom demandé n'existe pas dans notre base de données, nous vous répondons par un code 404.

GET
https://genderme.xyz/api?surname=luc
RESPONSE 200
{"surname": "luc", "gender": "male", "language": "fr"}
RESPONSE 404
Sorry, we don't know this surname :-(
RESPONSE 403
Quota exceeded! See you tomorrow...

Limites

Notre service est gratuit dans la limite de 10'000 requêtes par jour. Passé ce quota, vous recevrez une réponse 401.
